# Decocraft Interiors

> Craft in Every Corner

Official website for **Decocraft Interiors** — a young interior design studio crafting bespoke, meticulous and intentional spaces across Hyderabad.

## Stack
- Pure HTML / CSS / JS — no framework, no build step
- All assets self-contained in `index.html`
- Deployed on Vercel via GitHub

## Deploying
Push to `main` branch → Vercel auto-deploys instantly.

## Updating the site
1. Edit `index.html`
2. `git add . && git commit -m "your update" && git push`
3. Vercel picks it up in ~30 seconds

## Local preview
Just open `index.html` in any browser — no server needed.
